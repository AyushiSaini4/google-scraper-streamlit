# 🔍 Smart Web Search App

A simple and efficient web search application built using **Python, Streamlit, and BeautifulSoup**.  
This app allows users to search the web and view top results using DuckDuckGo (no API required).

---

## 🚀 Features

- 🔎 Search any query in real-time  
- ⚡ Fast and lightweight scraping  
- 🌐 Uses DuckDuckGo (no API key needed)  
- 🎯 Displays top 5 search results  
- 💻 Clean and interactive UI with Streamlit  

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- BeautifulSoup  
- Requests
